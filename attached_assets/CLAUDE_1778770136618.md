# CLAUDE CODE PREFERENCES

## Local Standalone Apps Template

When I request "local standalone apps" or "beginner-friendly apps", use this configuration:

### Project Type
- **Distribution**: Single HTML file (double-click to run)
- **Target Users**: Absolute beginners, zero technical knowledge
- **Platform**: Cross-platform (Windows, Mac, Linux)
- **Connectivity**: Offline-first, no backend required
- **Storage**: localStorage for persistence

### Tech Stack (Always Use)
- React 18 + TypeScript
- Vite with vite-plugin-singlefile
- Tailwind CSS
- shadcn/ui components
- Lucide React icons
- Inter font family

### Build Process
1. Configure `viteSingleFile()` plugin
2. Build to single HTML file
3. Test double-click functionality
4. Package with simple README

### Design Standards
- Choose design appropriate for app type and audience
- Implement responsive layouts suitable for content
- Add micro-interactions where they enhance UX
- Follow accessibility best practices

### File Structure
```
src/
├── components/ui/     # shadcn components
├── pages/            # Page components
├── lib/
│   ├── storage.ts    # localStorage utilities
│   └── utils.ts      # Common utilities
└── types.ts          # TypeScript definitions
```

### Commands to Run
- `npm install vite-plugin-singlefile`
- Configure vite.config.ts with viteSingleFile()
- `npm run build` → single HTML file output
