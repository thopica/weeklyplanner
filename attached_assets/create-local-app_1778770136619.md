# Quick Start Command for Local Standalone Apps

When starting a new project, use this prompt:

---

**"Create a local standalone app for [APP_PURPOSE] using my standard template:**

- **Target**: Absolute beginners, single HTML file distribution
- **Tech Stack**: React + TypeScript + Tailwind + shadcn/ui + Vite + vite-plugin-singlefile
- **Storage**: localStorage with custom utilities
- **Design**: Professional bento grids, Inter fonts, zinc/blue color scheme
- **Build**: Single HTML file output via `viteSingleFile()`
- **Distribution**: Double-click to run, works offline, cross-platform

**Features needed**: [LIST_SPECIFIC_FEATURES]

**Reference**: Use CLAUDE-TEMPLATE.md for architecture patterns, build config, and design standards."

---

This gives Claude Code all context upfront without re-explaining the entire tech stack each time.
